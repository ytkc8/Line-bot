## Usage
### Preparation
First, copy environment setting file from sample.

`$ cd /path/to/checkout/directory`

`$ cp -p .env.sample .env`

Second, input 'CHANNEL_TOKEN' and 'CHANNEL_SECRET' variables to '.env' file from line dev site.

Third, source '.env'.

`$ . .env`

### Getting start
`$ make start`
